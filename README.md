# RING

## Pre-requisites

Install uv

On macOS, Linux
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## Development

To setup the environment
```
uv venv
```

To run the ring app locally
```
sh ./run.sh
```

To install ring app in editable mode
```
rm -rf <SyftboxSyncDir>/apps/ring

syftbox install app .
```
